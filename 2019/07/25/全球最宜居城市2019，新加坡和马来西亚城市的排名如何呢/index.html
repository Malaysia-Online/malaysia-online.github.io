全球最宜居城市2019，新加坡和马来西亚城市的排名如何呢
发表于 2019-07-25| 更新于 2019-12-21|新加坡社会人文
阅读量:

 

FeiFei

国际HR顾问机构Mercer公布了2019年全球最宜居城市的排名榜，马来西亚的吉隆坡和新山，以及新加坡在亚洲区榜上有名哦！接来下，大家跟随“大马之声”一起往下看！

亚洲区排名点评

奥地利的维也纳位于排名榜的榜首。马来西亚的吉隆坡和新山也有不错的成绩，分别是第85和第101名。在所有的东南亚国家里，新加坡的排名最高，为第25名，其次就是我们的马来西亚。另外，亚洲区排名榜上还有日本的城市，包括东京、大阪、名古屋、神户，香港和韩国首尔也在排名榜中。亚洲地区排名垫底的城市是柬埔寨的金边（可能是因为地区的安全问题吧）。

排名参考因素

Mercer一共对全世界230多个国家和地区的450个城市进行了排名，主要参考以下一些因素：社会环境和政治环境、经济发展、娱乐休闲、房价、医疗、教育、交通、公共服务等。整个排名榜上，伊拉克的巴格达排最后。
