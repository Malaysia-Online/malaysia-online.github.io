马来西亚留学签证和陪读签证～办理攻略

  • 东南亚 留学移民
  • 2019-11-28

 

FeiFei

序

如今，有越来越多的中国学生开始出国留学，来新加坡和马来西亚留学的中国人也逐年增加。留学难免要碰到办理留学签证和陪读签证，其实办理这些签证并没有很多人想象的那么难！今天，FeiFei就带你了解一下，到底如何办理留学签证和陪读签证？下面就以马来西亚的留学签证和陪读签证作为例子说明！

留学签证办理

1）办理步骤

马来西亚的学生签证很容易办理，成功率很高，不用保证金，也不用面试，所需材料少，时间也比较快。大致可分为以下几个步骤：

（1）资料审核通过，交完费用，提交体验报告（研究生不用体验）；
（2）学校就会向马来西亚的EMGS (Education Malaysia Global Services)提交申请，帮你申请一份反签信Visa Approval Letter application (VAL)，时间周期为一个月-两个月；
（3）然后就可以用这个反签信 VAL，去马来西亚驻中国大使馆（北京、上海、广州、昆明都可以）办理签证了；
（4）不过，首次办理的马来西亚学生签证是单次入境签证Single Entry Visa application (SEV)，跟旅游签证差不多，有效期是3个月，凭SEV可以入境马来西亚；
（5）开学之前，先去学校报到注册缴费，同时在学校认可的任一马来西亚医院或私人诊所体检，并拿到健康证明；
（6）然后抓紧去学校的签证部门，提交所需材料，将单次签证换成一年有效期的正式学生签证（Student Pass application），学生签证可以多次出入境；
（7）之后每年续签一次，这样就避免了有些不法分子利用学生证进行非法打工。

2）学生签证办理所需材料

（1）学生本人的有效护照；
（2）护照大小照片若干；
（3）学校录取通知书；
（4）经公证的学历学位书；
（5）经公证的成绩单；
（6）个人简历；
（7）语言成绩证明；
（8）银行存款等资金证明；
（9）体检或健康证明等。

3）学生签证续签注意事项

（1）在学校保持80%以上的出勤率；
（2）在学校的成绩不能太差；
（3）对出勤率不足80%的学生，EMGS有权拒绝学生签证的续签；

4）关于学生签证费用

（1）马来西亚的学生签证可能不同学校和申请地区有些差异，但总体在1500马币上下；
（2）交给EMGS和学校的申请费是不退的，不过，假如毕业后你的签证期限没用完，EMGS会退一些保险费的。

陪读签证办理

1）陪读签证办理须知

（1）为了鼓励更多外国人来马来西亚留学，马来西亚政府采取了很多措施，包括允许学生的父母或者学生的配偶陪读，但陪读人员不能在马来西亚工作；
（2）如果学生有马来西亚第二家园的身份，可以读马来西亚任何中小学和大学。如果学生没有第二家园身份，只能去支持学生签证的学校读，而陪读人需要办理陪读签证，办理陪读签证只能是父母或配偶其中一人，学生年纪小的，最后是妈妈陪读，妈妈办理陪读签证容易成功；
（3）申请马来西亚陪读签证要提供存款证明，一般是15万马币左右，最好是长期存款。除了父母可以陪读之外，通过夫妻关系也可以陪读；
（4）学生签证一般1500马币左右，陪读签证费用为1000马币左右。

2）陪读签证所需材料

（1）陪读人护照复印件，记得复印整本护照，包括封面及所有空白页；
（2）学生护照复印件，复印整本护照，包括封面及所有空白页；
（3）陪读人的结婚证或离婚证，并经过公证。如果有离婚证书，则需提供孩子抚养权归属证明；
（4）陪读人护照照片（白色背景）若干张；
（5）学生与陪读人的亲属关系公证文件；
（6）工作或在职证明；
（7）银行存款证明；
（8）购买马来西亚的医疗保险，并提供复印件证明。

3）陪读注意事项

（1）遵守马来西亚当地的法律法规和生活习惯；
（2）贵重物品要随身携带或存放好；
（3）户外活动要主要安全，最好结伴出门；
（4）随身携带护照等身份证件；
（5）不要随身携带过多现金。

最后

随着出国留学越来越热，留学生越来越低龄化，陪读的需求也越来越多。关于马来西亚的留学签证和陪读签证，FeiFei就聊这么多，希望上面的内容对你有帮助哦！
