吉隆坡租房-Pantai Panorama公寓
发表于 2019-12-05| 更新于 2020-08-08|东南亚留学移民
阅读量:

 

FeiFei

序

Pantai Panorama公寓位于吉隆坡Bangsar South，是一个设计精良、环境优美的住宅小区。附近有商场KL-Gateway Mall、学校Universiti Malaya、地铁轻轨LRT，公交巴士站、的士站、医院LifeCare等，是一个既安静又方便且实惠的中高档小区。

房屋介绍

Pantai Panorama公寓，由马来西亚著名的房地产开发商Amcorp Properties Berhad于1996年建成，一共5栋公寓组成一片小区，每栋公寓约18层高，整个小区共700多个单元。小区内设施完善，超市、餐厅、健身房、游泳池、SPA泳池、羽毛球场、停车场等配套完善。24小时保安服务，安全舒适。

Pantai Panorama小区背靠吉隆坡森林生态区，四周安静、绿色环保，是一个理想的居住地，也是很多人梦寐以求的生活环境。除了每日营业的超市和餐厅，每个公寓还配有泳池、健身房，有些公寓还配有烧烤区，每个公寓楼下都有足够的停车位。

距离马来亚大学5分钟车程，离Mid Valley 也没几公里；附近地铁轻轨LRT、公交巴士、出租车都很方便，步行10分钟左右即可到达。

屋内配套设备

Pantai Panorama公寓室内配套设备齐全，包括沙发、洗衣机、冰箱、空调、电扇、电视机、衣柜、橱柜、桌椅、电器等。

租房价格

Pantai Panorama有适合单人居住和家庭居住的不同房型，所以租房价格也不一样，以下分别说明：

1）一室一厅一卫一厨 1200-1900RM/月，有些单元配有停车位。（适合1-2人居住，也适合情侣居住）

2）三室一厅两卫一厨2500-3200RM/月，部分单元配有停车位（适合3-5人合租，或家庭租住）。

想要租房的朋友们，可以联系FeiFei，微信号为：angel-emmy。
