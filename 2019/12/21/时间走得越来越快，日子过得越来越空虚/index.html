时间走得越来越快，日子过得越来越空虚
发表于 2019-12-21| 更新于 2020-10-25|专题生活之南
阅读量:

 

林子

序

众里寻他千百度
蓦然回首
2019已到头
时光飞逝
转眼间又到年底
回想过去一年
匆匆忙忙
好像做了很多事
又好像什么都没做
虽然年龄渐长
但日子过得
已经没有了儿时的充实感
百思不解
为什么时间走得越来越快？
如何才能让日子过得慢一点？

不是每天都有新体验

心理学家曾说
童年的时候
新鲜事很多
每天都有新的体验
有很多值得纪念的第一次
第一次上学
第一次恋爱
但长大成人后
新体验已不多
第一次都被用完了
日子都是重复着过
所以
每天每周
甚至每个月每一年
都感觉
时光流逝
人越来越空虚

压迫感让生活慢不下来

科技越来越先进
文明越来越高
可生活却越来越忙
每天有太多的事要做
有很多的人要见
时间总是不够用
沉重的压迫感
推着时间加速流逝
因此
生活总是过得很匆忙
怎么也慢不下来
空虚感也不断加重

多接触新事物

减少空虚感
远离时间压迫
可以多尝试新事物
从点滴做起
周末去逛新商场
假期去郊外探险
只要能带来全新体验
远离按部就班
空虚就会减少
日子也会慢下脚步

多给自己留一点空间

每天的生活
都是重复
很容易陷入惯性
进入自动导航模式
多给自己一点空间吧
去创造属于你的记忆
可以尝试晨间冥想
昏后散步
不忘提醒自己
生活再忙
也要留时间给自己
经验再多的老司机
也不能一直踩油门
该刹车的时候
绝不心软

多学习

如果希望日子过得慢一点
可以尝试多学习新事物
将心思投入学习
无论是
学习新技能还是新语言
都可以让我们回到初心
回到孩提时代
单纯地理解新事物
获得更多知识
感受与时光擦肩而过

最后

如果有一天
你感觉
时间在加速
日子变空虚
生活很孤单
请记得
给自己留一点空间
接触新事物
多学习
保持单纯
不忘初心
